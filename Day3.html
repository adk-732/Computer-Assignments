Setting up XAMPP on Your PC

Follow these steps to set up XAMPP on your PC:

Step 1: Download and Install XAMPP

Download the XAMPP installer from the official website and follow the installation instructions.

Step 2: Create a New Folder for Your Projects

Create a new folder to store your projects, for example, 'Test Folder'.

Step 3: Create a New PHP Page

Create a new PHP page using the following code:

```php PHP-Test Hello World

'; ?> ```

Save this file as 'test.php' in your 'Test Folder'.

Step 4: Open Your PHP Page in a Web Browser

Open your web browser and navigate to 'localhost/test/test.php'. If you see the 'Hello World' message, then you've successfully installed and configured XAMPP.
