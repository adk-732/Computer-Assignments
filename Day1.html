Database Management System's Basic Terminology

Data

Data refers to the facts and statistics collected together for reference or analysis.

Database

A database is a collection of organized data that is stored in a way that allows for efficient retrieval and manipulation.

DBMS

A DBMS (Database Management System) is a software system that allows you to define, create, maintain, and manipulate databases.

Schema

A schema is the overall structure or organization of a database, including the relationships between different data entities.

Table

A table is a collection of related data entries, with each entry consisting of several fields or columns.

Tuple

A table is, as the name suggests, a tabular body in which the related data is filled in. The table is first filled in with the content's heading using the top row of the table. And then, the rest of the data is filled in accordance to the row headings.

Row

A row is a single entry in a table, consisting of several fields or columns.

Column

A column is a single field or attribute of a table, containing specific information about each entry.

Primary Key

A primary key is a unique identifier for each entry in a table, used to distinguish it from other entries.

Foreign Key

A foreign key is a field in a table that refers to the primary key of another table, establishing a relationship between the two tables.
